# Startup Idea Generator 🚀

A simple web application built using **Python and Flask** that generates random startup ideas.

This project was created while learning basic web development with Flask and demonstrates how backend logic can dynamically generate content for a webpage.

---

## Features

* Generates random startup ideas
* Interactive **Generate Idea** button
* **Copy Idea** feature to copy the generated idea
* Clean and simple user interface

---

## Tech Stack

* **Python**
* **Flask**
* **HTML**
* **CSS**
* **JavaScript**

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/startup-idea-generator.git
```

2. Navigate to the project folder

```
cd startup-idea-generator
```

3. Install Flask

```
pip install flask
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
